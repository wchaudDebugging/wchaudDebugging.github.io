About

I build practical, interactive tools to help people understand algorithms — from pathfinding to sorting. Below you'll find demos that run in your browser: maze pathfinding (BFS / DFS / Dijkstra), a sorting visualizer, and a sorting race to compare algorithms.

Maze Pathfinder

Toggle walls by clicking the grid. Run a single algorithm at a time.

Complexities:
  • BFS: O(V+E) time, O(V) space
  • DFS: O(V+E) time, O(V) space
  • Dijkstra: O(E log V) time, O(V) space
